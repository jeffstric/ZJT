# 图片涂色编辑功能

## 功能概述

图片涂色编辑功能允许用户在图片节点中直接对图片进行涂色修改，涂色后的图片将直接替换原图。适用于给黑白图上色、修改图片局部颜色、添加标记等场景。

## 使用方法

1. 在图片节点中上传或生成图片
2. 点击"涂色编辑"按钮打开涂色编辑器
3. 使用画笔在图片上涂色
   - 调整画笔大小（5-50px）
   - 选择颜色
   - 调整透明度（0-100%）
4. 支持撤销和清空操作
5. 点击"确认"按钮
6. 涂色后的图片自动上传并替换节点中的原图

## 技术实现

### 文件结构

- `web/video_workflow.html` - 涂色编辑模态框HTML结构
- `web/css/video_workflow.css` - 涂色编辑器样式
- `web/js/image_coloring_editor.js` - 涂色编辑器核心模块
- `web/js/nodes.js` - 图片节点集成

### 工作流程

1. **打开编辑器**：加载原始图片到Canvas
2. **用户涂色**：在Canvas上直接绘制/修改颜色
3. **上传涂色图**：将涂色后的Canvas转为图片并上传到服务器
4. **替换原图**：涂色后的图片直接替换节点中的原图
5. **自动保存**：自动保存工作流

### API接口

使用 `/api/video-workflow/upload` 接口上传涂色后的图片：

```
POST /api/video-workflow/upload
Content-Type: multipart/form-data
Headers:
  Authorization: {auth_token}
  X-User-Id: {user_id}

参数：
- file: 涂色后的图片文件（PNG格式）

返回：
{
  "code": 0,
  "data": {
    "url": "涂色后的图片URL"
  },
  "message": ""
}
```

### 核心功能

#### Canvas绘制
- 支持鼠标和触摸事件
- 可调节画笔大小、颜色、透明度
- 撤销功能（最多20步历史记录）
- 清空画布功能

#### 图片处理
- 自动适配图片尺寸
- 将Canvas转换为PNG格式
- 上传到服务器获取URL
- 直接替换节点中的图片

## 注意事项

1. 涂色编辑需要先有图片（上传或生成）
2. 上传过程中会禁用涂色按钮，防止重复提交
3. 涂色后的图片会直接替换原图，原图不会保留
4. 编辑成功后自动保存工作流
5. 涂色是在原图上直接绘制，不是AI生成新图
