# 自动提交数据库功能

## 功能概述

剧本智能创作系统 (`script_writer.html`) 支持自动提交数据库功能，开启后系统每分钟自动提交暂存数据到数据库。

## 功能位置

位于页面顶部导航栏右侧，"提交数据"按钮旁边。

## 使用方式

### UI 控件

| 控件 | 说明 |
|------|------|
| 🔄 自动提交开关 | Toggle 开关，开启/关闭自动提交 |
| 倒计时显示 | 开启时显示下次提交的剩余时间 |

### 操作步骤

1. 选择提交间隔（默认1分钟）
2. 打开"🔄 自动"开关
3. 系统开始倒计时，到达时间后自动提交
4. 提交完成后自动调度下一次提交

## 随机延迟策略

为避免多用户同时提交造成系统压力，采用随机延迟策略：

- **基础间隔**: 用户选择的分钟数（如1分钟 = 60秒）
- **随机偏移**: 0-30秒的随机延迟
- **实际间隔**: 基础间隔 + 随机偏移

例如：选择1分钟间隔，实际提交间隔为 60-90 秒。

## 设置持久化

用户设置通过 Cookie 保存，有效期1年：

- 开关状态（开启/关闭）

页面刷新后会自动恢复用户设置。

## 技术实现

### 核心函数

| 函数 | 说明 |
|------|------|
| `toggleAutoSubmit()` | 切换自动提交开关 |
| `startAutoSubmit()` | 启动自动提交 |
| `stopAutoSubmit()` | 停止自动提交 |
| `scheduleNextSubmit()` | 调度下次提交（含随机延迟） |
| `updateCountdownDisplay()` | 更新倒计时显示 |
| `saveAutoSubmitSettings()` | 保存设置到 Cookie |
| `loadAutoSubmitSettings()` | 从 Cookie 加载设置 |
