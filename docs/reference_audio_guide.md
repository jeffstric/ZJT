# 参考音频功能说明

## 概述

参考音频功能允许用户为对话组节点和角色库上传参考音频，用于生成更准确的语音合成效果。系统会优先使用对话组节点中的参考音频，如果未找到匹配的角色音频，则会从角色库中查找。

## 功能特性

### 1. 对话组节点参考音频

对话组节点支持上传多个参考音频文件，每个音频关联一个角色名称。

**限制条件：**
- 最多支持 6 个参考音频
- 每个音频文件大小不超过 10MB
- 每个音频时长不超过 20 秒

**使用步骤：**

1. 打开对话组节点
2. 在"参考音频"区域点击"添加音频"按钮
3. 选择音频文件（系统会自动验证文件大小和时长）
4. 输入该音频对应的角色名称
5. 上传成功后，音频会显示在列表中

**音频匹配规则：**
- 生成对话音频时，系统会根据对话的角色名称匹配参考音频
- 角色名称必须完全匹配（区分大小写）
- 如果对话组中找到匹配的参考音频，将优先使用
- 如果未找到匹配，则从角色库中查找该角色的参考音频

### 2. 角色库参考音频

在创建或编辑角色时，可以为角色上传参考音频。

**限制条件：**
- 每个音频文件大小不超过 10MB
- 每个音频时长不超过 20 秒

**使用步骤：**

1. 打开角色创建/编辑界面
2. 在"参考音频"字段选择音频文件
3. 系统会自动验证文件大小和时长
4. 验证通过后，音频会显示预览播放器
5. 保存角色后，参考音频会关联到该角色

## 音频生成优先级

当生成对话音频时，系统按以下优先级查找参考音频：

1. **对话组节点参考音频**（最高优先级）
   - 根据对话的角色名称在对话组节点的参考音频列表中查找
   - 如果找到匹配的角色音频，使用该音频

2. **角色库参考音频**（次优先级）
   - 如果对话组中未找到匹配的参考音频
   - 系统会从当前世界的角色库中查找该角色
   - 使用角色库中配置的参考音频

3. **无参考音频**
   - 如果以上两种方式都未找到参考音频
   - 生成音频可能会失败或使用默认音色

## 使用场景

### 场景1：临时角色或未知角色

如果剧本中出现了角色库中不存在的角色，可以在对话组节点中临时添加该角色的参考音频，避免音频生成失败。

**示例：**
```
剧本中有"路人甲"这个角色，但角色库中没有创建该角色。
可以在对话组节点中添加一个参考音频，角色名设置为"路人甲"。
```

### 场景2：同一角色不同情感

同一个角色在不同场景中可能需要不同的音色表现，可以在不同的对话组节点中为同一角色上传不同的参考音频。

**示例：**
```
"小明"在场景A中情绪激动，在场景B中情绪平静。
可以在场景A的对话组中上传激动的参考音频。
在场景B的对话组中上传平静的参考音频。
```

### 场景3：快速测试音色

在正式创建角色之前，可以先在对话组节点中测试不同的参考音频效果，找到最合适的音色后再创建角色。

## 注意事项

1. **音频格式**：支持常见的音频格式（如 mp3, wav, m4a 等）
2. **音频质量**：建议使用清晰、无背景噪音的音频文件
3. **角色名称**：必须与对话中的角色名称完全一致
4. **文件限制**：严格遵守 10MB 和 20 秒的限制，超出限制的文件会被拒绝
5. **存储位置**：上传的音频会保存到服务器，重新加载工作流时会自动恢复

## 常见问题

**Q: 为什么我的音频上传失败？**
A: 请检查音频文件是否超过 10MB 或时长超过 20 秒。

**Q: 角色名称不匹配怎么办？**
A: 确保对话组中的参考音频角色名称与对话中的角色名称完全一致（包括大小写）。

**Q: 可以修改已上传的参考音频吗？**
A: 可以删除后重新添加新的参考音频。

**Q: 对话组参考音频和角色库参考音频有什么区别？**
A: 对话组参考音频优先级更高，适合临时使用或特殊场景；角色库参考音频是全局的，适合常用角色。
