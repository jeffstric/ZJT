========================================
智剧通 Windows 启动文件说明
========================================

【启动文件】
----------------------------------------

1. 点我启动.exe（推荐）
   - 双击即可启动服务
   - 在系统托盘显示启动状态图标
   - 启动过程中会显示气泡提示（正在启动MySQL...等）
   - 服务就绪后自动打开浏览器
   - 右键托盘图标可查看日志或退出

2. start_silent.vbs
   - VBS 脚本，静默启动（备用方案）
   - 不显示托盘图标

3. start.bat
   - 批处理脚本，显示详细日志
   - 适合调试或查看启动过程

【启动流程】
----------------------------------------
点我启动.exe / start_silent.vbs / start.bat
    ↓
start_windows.py（Windows 启动管理器）
    ↓
1. 检查并更新配置文件（ffmpeg 路径等）
2. 启动 MySQL 服务
3. 首次运行时自动初始化数据库
4. 启动 Web 服务（run_prod.py）
5. 自动打开浏览器
6. 持续监控服务状态

【配置文件】
----------------------------------------
- config_prod.yml：生产环境配置
- 首次启动时从 config.example.yml 自动创建
- ffmpeg/ffprobe 路径会自动更新为当前目录

【注意事项】
----------------------------------------
- 确保 bin/mysql 目录存在（MySQL 服务）
- 确保 bin/ffmpeg 目录存在（视频处理）
- 关闭窗口或按 Ctrl+C 停止服务

========================================
推荐使用"点我启动.exe"，简单方便！
========================================
